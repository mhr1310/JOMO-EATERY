# Jomo Eatery Kuala Nerus - Official Website

A modern, responsive café website built with HTML5 and CSS3 only.

## About

Jomo Eatery is a popular café in Kuala Nerus offering Western food, desserts, pastries, and specialty coffee. This website serves as the official online presence for customers and visitors.

## Features

- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ 6 complete pages (Home, About, Menu, Reservation, Gallery, Contact)
- ✅ Modern minimalist aesthetic
- ✅ Clean café-inspired color palette
- ✅ Professional food photography
- ✅ No JavaScript required - Pure HTML5 & CSS3
- ✅ Works offline - No dependencies

## Pages

1. **Home** - Hero section, introduction, featured specialties
2. **About** - Café story, mission, values, target audience
3. **Menu** - Western Food, Desserts & Pastries, Drinks & Coffee
4. **Reservation** - Booking form with all required fields
5. **Gallery** - Photo grid showcasing café ambiance and food
6. **Contact** - Location, hours, phone, email, Google Maps

## How to Use

1. Open `index.html` in any modern web browser
2. Navigate between pages using the top navigation menu
3. All features work without requiring a server
